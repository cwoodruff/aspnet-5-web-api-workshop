RESPONSE CACHING

Add to ConfigureServices in Startup.cs -- services.AddResponseCaching();

Add to Configure in Startup.cs -- app.UseResponseCaching();
** Note: needs to go after CORS

Add to Controllers or Actions -- [ResponseCache(Duration = 604800)]
** Note: 604800 is seconds and equals a week

What is seen in the Response Header
Response Header:
Cache-Control: public,max-age=604800

Test in Postman

IN-MEMORY CACHING

Install Microsoft.Extensions.Caching.Abstractions NuGet package to Domain

Add to ConfigureServices in Startup.cs -- services.AddMemoryCache();

Add code to Supervisor as needed

DISTRIBUTED CACHING
Install tool for MSSQL distrubuted caching using the commend in Command Prompt -- "dotnet tool install --global dotnet-sql-cache"

Run from Command Prompt -- "dotnet sql-cache create "Data Source=.;Initial Catalog=ChinookCacheDb;Integrated Security=True;" dbo ChinookCache"

Add Microsoft.Extensions.Caching.SqlServer NuGet package to Domain

Add to ConfigureServies in Startup.cs -- services.AddDistributedMemoryCache();

Add code to Supervisor as needed
